# Курируемые источники для Layer-2 карточек

Одна папка = один ЖК: `sources/<slug>/` (slug — как в complexes.jsonl).
Внутри — текстовые файлы `.txt`, по одному на источник. Формат файла:

```
url: https://tengrinews.kz/...
title: Название статьи/страницы
type: news | reviews | forum | official
date: 2016-05-17        (если известна)
---
Сюда вставляете скопированный текст источника (или его значимую часть).
```

## Как собирать (20-30 мин на ЖК)
1. Поиск: «ЖК <название> Астана отзывы / проблемы / затопило / трещины /
   застройщик суд / задержка сдачи».
2. 2GIS и Яндекс.Карты: страница дома → вкладка «Отзывы» → скопировать
   тексты отзывов (5-15 штук достаточно).
3. Новости: tengrinews, informburo, krisha.kz/content — всё, где ЖК
   упоминается по имени.
4. Официальное: реестры проблемных объектов, разрешения — type: official.

## Правила (их же enforce'ит generate_cards_l2.py)
- Не редактируйте тексты — вставляйте как есть, мусор модель отфильтрует.
- Один источник = один файл. НЕ склеивайте.
- Тексты с именами частных лиц можно вставлять — в карточку имена не
  попадут (запрещено промптом и проверяется на выходе).

Градация достоверности, которую получит карточка:
- **confirmed** — официальный документ или новость СМИ (type: official/news);
- **reported** — жалоба/утверждение минимум из ДВУХ независимых
  источников-отзывов;
- **rumor** — единичное неподтверждённое упоминание;
- **no_data** — тема не покрыта собранными источниками.
